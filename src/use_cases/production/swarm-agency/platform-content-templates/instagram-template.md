## Format Specifications

### Instagram Reels

**Technical Specs:**
- **Aspect Ratio**: 9:16 (vertical)
- **Resolution**: 1080x1920 minimum
- **Duration**: 15-90 seconds (sweet spot: 30-60 seconds)
- **Frame Rate**: 24fps, 30fps, or 60fps
- **File Format**: MP4 or MOV
- **File Size**: Under 500MB

**Content Strategy:**
- Similar to TikTok but with Instagram-specific audience
- More polished/produced than TikTok (generally)
- Focus on aesthetics + value
- Use trending audio when relevant

**Hook Templates:**
```
// Value-driven
"Stop scrolling if you struggle with {problem}"

// Educational
"Here's the {number}-step framework for {outcome}"

// Story-driven
"I {action} for {time period} and here's what happened"

// Visual hook
[Start with visually striking imagery before text]
```

### Instagram Carousels

**Technical Specs:**
- **Aspect Ratio**: 1:1 (1080x1080) or 4:5 (1080x1350)
- **Number of slides**: 7-10 slides (optimal)
- **File Format**: JPG or PNG
- **File Size**: Under 30MB per slide

**Content Structure:**
```
Slide 1: Hook/Headline (make them want to swipe)
Slide 2-8: Educational content (one point per slide)
Slide 9: Summary or "cheat sheet"
Slide 10: CTA (save this, link in bio, follow for more)
```

**Design Best Practices:**
- Consistent branding throughout
- 40pt+ font for readability
- Minimal text per slide (don't overwhelm)
- High contrast colors
- Include visual variety (charts, icons, photos)

**Carousel Formulas:**

```
// List-style
"5 {Topic} Tips You Need to Know"
→ Slide 1: Hook
→ Slides 2-6: One tip per slide
→ Slide 7: CTA

// Step-by-step
"How to {Achieve Outcome} in {Number} Steps"
→ Slide 1: Hook
→ Slides 2-5: Steps with visuals
→ Slide 6: CTA

// Before/After
"Transform Your {Area} with This Strategy"
→ Slide 1: Before state
→ Slides 2-4: Process
→ Slide 5: After state
→ Slide 6: CTA

// Myth-busting
"3 {Topic} Myths That Are Holding You Back"
→ Slide 1: Hook
→ Slides 2-4: Myth → Reality
→ Slide 5: CTA
```

### Instagram Stories

**Technical Specs:**
- **Aspect Ratio**: 9:16 (1080x1920)
- **Duration**: Each story up to 15 seconds (clips for longer)
- **File Format**: JPG, PNG, MP4
- **File Size**: Under 30MB per story

**Story Series Formula:**
```
Story 1: Hook (make them tap through)
Story 2-4: Value/content delivery
Story 5: Interactive element (poll, question, quiz)
Story 6: CTA (reply, swipe up, link sticker)
```

**Interactive Elements:**
```
// Polls (engagement)
"Which tip resonates most? [Option A] or [Option B]"

// Questions (feedback)
"What's your biggest challenge with {topic}?"

// Quizzes (education)
"True or False: {Statement about topic}"

// Countdowns (urgency)
"New {content/product} drops in [time]"

// Link stickers (traffic)
"Get the full guide → [link sticker]"

// Sliders (opinion)
"How much do you agree with this statement?"
```

**Story Highlights Strategy:**
- Organize by topic/category
- Use consistent cover designs
- Update weekly with best-performing stories
- Categories: About, Services, Products, Testimonials, Education, Behind the Scenes

### Single Image Posts

**Technical Specs:**
- **Aspect Ratio**: 1:1 (1080x1080) or 4:5 (1080x1350)
- **File Format**: JPG or PNG
- **File Size**: Under 30MB

**When to Use:**
- High-quality photography
- Infographics
- Quotes (with design)
- Behind-the-scenes moments
- User-generated content

**Caption Formula:**
```
Line 1: Hook (make them stop scrolling)
Line 2-3: Value or story (expand on hook)
Line 4: Example or proof
Line 5: CTA
```

## Hashtag Strategy

### Instagram Hashtag Formula (15-30 total)

```markdown
1. **Super Broad** (3-5 tags)
   - 1M+ posts using
   - Example: #marketing, #business, #entrepreneur

2. **Broad/Niche** (5-10 tags)
   - 100K-1M posts using
   - Example: #digitalmarketing, #smallbiztips, #contentcreator

3. **Specific** (5-10 tags)
   - 10K-100K posts using
   - Example: #emailmarketingtips, #canvatemplates, #reelsstrategy

4. **Hyper-Specific/Branded** (3-5 tags)
   - Under 10K posts using
   - Example: #yourbrand, #yourbrandedhashtag
```

### Hashtag Placement Options

```
Option 1: First comment (cleaner caption)
Option 2: Bottom of caption (all in one place)
Option 3: Weaved into caption (more natural)

Recommended: First comment for better caption readability
```

### Hashtag Research Strategy

1. **Check competitors** in your niche
2. **Use Instagram's search** for suggestions
3. **Test hashtag sets** (track which perform)
4. **Rotate hashtags** (don't use same set every post)
5. **Mix large + small** for reach + targeting

## CTA Strategies

### Engagement CTAs

```
- "Save this for later" (highlights value)
- "Share this with someone who needs it" (extends reach)
- "Tag a friend who would love this" (boosts engagement)
- "Double tap if you agree" (drives likes)
- "Comment '{emoji}' if you want Part 2" (sparks conversation)
- "Which tip resonates most? 1, 2, or 3?" (encourages comments)
```

### Growth CTAs

```
- "Follow for more {topic} content" (clear value proposition)
- "Turn on post notifications so you never miss a tip" (lifetime value)
- "Check out my highlighted stories for {topic}" (drives to highlights)
```

### Traffic CTAs

```
- "Link in bio to {resource}" (clear direction)
- "DM me '{keyword}' for {value}" (creates conversation)
- "Click the link in my bio for the full guide" (specific)
```

## Algorithm Optimization

### Key Ranking Signals (2025)

1. **Engagement Rate** (Most Important)
   - Comments > Saves > Shares > Likes
   - Focus on sparking genuine conversation

2. **Watch Time** (For Reels)
   - Aim for 100% completion
   - Rewatches signal quality

3. **Session Time**
   - Keep viewers on platform longer
   - Create binge-worthy content

4. **Relationship**
   - Engage with your audience
   - Reply to comments (especially first hour)

5. **Timeliness**
   - Post when audience is active
   - Stay consistent with posting schedule

### Content Categorization

```
Help Instagram categorize your content:

1. Use consistent hashtags (mix of broad + specific)
2. Stay in your niche (don't pivot randomly)
3. Use keywords in captions (SEO matters)
4. Create content series (builds authority)
5. Engage with similar accounts
```


## Performance Benchmarks

### Good Engagement Rates (by follower count)

```
Under 1K followers: 5-8% engagement rate
1K-10K followers: 3-5% engagement rate
10K-100K followers: 2-3% engagement rate
100K+ followers: 1-2% engagement rate

Engagement Rate Formula:
((likes + comments + saves) / followers) × 100
```

### Viral Content Indicators

```
Reel Views:
- Great: 5-10K views
- Viral: 50K+ views
- Super viral: 100K+ views

Feed Post Engagement:
- Great: 10%+ engagement rate
- Viral: 20%+ engagement rate
- Super viral: 50%+ engagement rate

Saves (strongest signal of value):
- Great: 100+ saves
- Viral: 500+ saves
- Super viral: 1,000+ saves
```

## Common Mistakes to Avoid

1. **Inconsistent branding** - Confuses audience and weakens brand
2. **Buying followers** - Hurts engagement rate and credibility
3. **Over-posting** - Quality > quantity (especially in feed)
4. **Ignoring stories** - Stories build relationships and trust
5. **Weak CTAs** - Every post should have a purpose
6. **Not engaging** - Reply to comments, especially early on
7. **Hashtag spam** - Use relevant, targeted hashtags only
8. **Poor lighting/quality** - Visual platform requires visual quality
