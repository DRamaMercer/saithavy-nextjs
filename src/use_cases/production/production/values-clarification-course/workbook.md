# Values Clarification Workbook

## Welcome to Your Values Journey

This workbook is your companion for the 5-day Values Clarification Course.

Each day has exercises designed to help you discover, test, and live your values.

**Instructions:**
- Complete each day's exercises before reading the next day's email
- Be honest with yourself—this isn't about who you wish you were, but who you actually are
- Give yourself time to reflect—don't rush through these exercises
- Write down your answers—thinking about them is different from writing them

### Values Decision Framework

Use this framework when making decisions:

**When facing a decision, ask:**

1. **Which values are relevant?**
   - Which of my top 5 values are involved?

2. **Which option honors my values?**
   - Which choice aligns with what matters to me?

3. **Can I live with the values compromise?**
   - If there's a values conflict, which will I prioritize?

4. **What would future me be proud of?**
   - 6 months from now, which choice will I respect?

## Bonus: Values Deck Creation

### Creating Your Physical Values Deck

**Materials needed:**
- Index cards or cardstock
- Pen or marker
- Something to carry your cards (wallet, bag, pocket)

**Instructions:**

1. **Write your top 5 values on separate cards**
   - One value per card
   - Write the value name large and clear
   - Add a brief definition or example if helpful

2. **Carry them with you**
   - Put them in your wallet, bag, or pocket
   - Keep them accessible throughout the day

3. **Review daily**
   - Morning routine: Read through your values
   - Before decisions: Pull out relevant cards
   - Evening reflection: How did you live your values today?

4. **Use as a decision-making compass**
   - When facing a choice, consult your values deck
   - Ask: "Which values are relevant here?"
   - Choose the option that honors your values

5. **Revisit quarterly**
   - Values evolve over time
   - Schedule a quarterly review
   - Update your deck if values have shifted

## Additional Resources

**Daily Values Check-In:**

Each morning, ask:
- Which value will I focus on today?
- What's one way I can live this value today?

Each evening, ask:
- How did I live my values today?
- Where did I violate my values?
- What can I do differently tomorrow?

**Values in Relationships:**

- Share your values with people close to you
- Ask about their values
- Discuss where your values align and conflict
- Use values to navigate disagreements

**Values at Work:**

- Identify which values are honored in your current work
- Identify which values are violated
- Look for roles or companies that align better
- Set boundaries to protect your values

**Values in Tough Times:**

- Return to your values deck for guidance
- Make decisions based on your values, not fear
- Remember: values are a compass, not a map
- Trust your values even when the path is unclear
