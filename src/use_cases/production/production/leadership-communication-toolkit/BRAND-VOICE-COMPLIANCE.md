# Brand Voice Compliance Check

**Leadership Communication Toolkit**

## Brand Voice Requirements

### Voice: Confident + Empathetic

**Confident:** Direct, clear, authoritative but not arrogant
- Uses declarative statements: "Here's what works"
- Shares principles as learned truths: "Don't wait to have difficult conversations"
- Provides specific scripts, not vague suggestions
- Admits mistakes to build trust, not to undermine authority

**Empathetic:** Warm, understanding, human
- Acknowledges emotions: "I can see this is difficult"
- Validates experiences: "Your frustration is valid"
- Shares vulnerability: "I've made every mistake in this toolkit"
- Focuses on growth and support, not criticism
