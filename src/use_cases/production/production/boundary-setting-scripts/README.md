## How to Use This Collection

### If You're New to Boundaries

**Start here:**
1. Read the **Progressive Practice Guide** - understand the approach
2. Choose 2-3 "easy" scripts that feel doable this week
3. Practice saying them out loud (it WILL feel weird - that's normal)
4. Use one in a real conversation
5. Reflect on how it went

**Your first week:**
- Pick ONE low-stakes boundary to set
- Practice the script 3-5 times
- Have the conversation
- Note what worked, what didn't
- Celebrate that you tried (even if it was messy)

### If You Know What You Need

**Go directly to:**
- **Context scripts** for your specific situation
- **Repair protocols** if boundaries were crossed
- **Conversation frameworks** for difficult talks
- **Cultural guide** for nuanced situations

**Quick access:**
- Use the **Quick Reference Guide** to find scripts fast
- Scripts are organized by relationship, difficulty, and scenario
- Each script includes: difficulty rating, when to use, what to expect

### If You're Struggling with Guilt

**You're not alone:**
- Read **Conversation Frameworks** - the "why" helps
- Start with **Progressive Practice** - build the muscle gradually
- Review **Boundary Repair** - for when it doesn't go perfectly
- Remember: Guilt ≠ You did something wrong

## Progressive Difficulty System

Scripts are rated by difficulty:

**🟢 Easy (Start Here)**
- Low stakes
- Low emotional charge
- People likely to be understanding
- Good practice opportunities

**🟡 Medium (Building Muscle)**
- Moderate stakes
- Some emotional charge
- People might push back mildly
- Requires some courage

**🔴 Hard (Advanced Practice)**
- High stakes
- High emotional charge
- People likely to resist strongly
- Significant relationship impact
- May need support system

**Recommendation:** Start with 🟢, practice until you feel confident, then move to 🟡. 🔴 scripts often need preparation and support.
