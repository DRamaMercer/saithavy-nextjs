## Day-by-Day Overview

### Day 1: What Are Values?
**Topic:** Values discovery through card sort
**Exercise:** 3-round values selection (75 values → top 5)
**Key Learning:** Values are what you ARE, not what you wish you were
**Time Required:** 20-30 minutes

### Day 2: Test Your Values
**Topic:** Reality testing — are values real or aspirational?
**Exercise:** Values audit (calendar + spending review)
**Key Learning:** Your values are what you DO, not what you SAY
**Time Required:** 30-40 minutes

### Day 3: Values in Action
**Topic:** How to live values through daily behaviors
**Exercise:** Values in practice worksheet (boundaries, relationships, actions)
**Key Learning:** Specific values = clear action. Vague values = no action.
**Time Required:** 40-50 minutes

### Day 4: Values Alignment Assessment
**Topic:** Mapping values across 6 life domains
**Exercise:** Values alignment map (career, relationships, health, growth, finances, community)
**Key Learning:** Misalignment costs energy, satisfaction, and wellbeing
**Time Required:** 45-60 minutes

### Day 5: Living Your Values
**Topic:** 30-day commitment and decision framework
**Exercise:** Values commitment + decision framework + values deck creation
**Key Learning:** Values are a compass, not a map—they guide but don't dictate
**Time Required:** 30-40 minutes

## Common Participant Questions

### "What if I can't narrow to 5 values?"
**Answer:** That's common. Some people have 6-8 core values. That's okay. Try to get to your top 3-5 that are non-negotiable.

### "What if I don't see 'my' value on the list?"
**Answer:** The list of 75 covers most common values, but you might have something unique. Write it down and add it to your list.

### "What if my values feel 'wrong' or 'bad'?"
**Answer:** There are no wrong values. Some values (like "Approval" or "Comfort") get judged. But they're real values for many people. Own what's true.

### "How often should I revisit my values?"
**Answer:** Do a formal review quarterly. Values can shift as you grow. That's not failure—it's evolution.

### "What if my values conflict?"
**Answer:** They will. That's normal. You'll need to prioritize based on the situation and season of life. No value can be honored 100% of the time.
