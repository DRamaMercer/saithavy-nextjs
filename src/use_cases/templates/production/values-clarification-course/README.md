# Values Clarification Course

**A 5-day email course + workbook to discover, test, and live your core values**

---

## Course Overview

This comprehensive course guides participants through a deep exploration of their personal values system over 5 days. Each day includes:

- **Email Lesson:** Insightful teaching on values with real examples
- **Workbook Exercise:** Interactive exercises to apply the learning
- **Actionable Takeaway:** Specific steps to implement immediately

4. **Day 4: Values Alignment** — Life domains assessment to find alignment gaps

## Course Structure

### Day 1: Discovery
**Goal:** Identify your top 5 core values from a comprehensive list

**Exercise:** Values Card Sort
- Round 1: Choose top 20 from 75 values
- Round 2: Narrow to top 10
- Round 3: Select final top 5 core values

**Key Learning:** Values aren't what you wish they were—they're what they actually are

### Day 2: Reality Testing
**Goal:** Test if your values are real or aspirational

**Exercise:** Values Audit
- Calendar review: How you spend your time
- Spending review: How you spend your money
- Recent examples: When did you live/violate each value?
- Honesty assessment: Real vs. aspirational

**Key Learning:** Your values are what you do, not what you say

### Day 3: Action Planning
**Goal:** Create specific plans for living each value

**Exercise:** Values in Practice
- What does living this value look like daily?
- What boundaries protect this value?
- What relationships support this value?
- What conflicts with this value?
- One action to take this week

**Key Learning:** Vague values = no action. Specific values = clear action.

### Day 4: Alignment Assessment
**Goal:** Map values across 6 life domains

**Exercise:** Values Alignment Map
- Career: Which values are honored/violated?
- Relationships: Which values are honored/violated?
- Health: Which values are honored/violated?
- Personal Growth: Which values are honored/violated?
- Finances: Which values are honored/violated?
- Community: Which values are honored/violated?

**Key Learning:** Misalignment has a cost—guilt, resentment, burnout

### Day 5: Commitment
**Goal:** Create a 30-day values commitment

**Exercise:** Living Your Values
- Choose 2-3 values to focus on
- One behavior to start
- One behavior to stop
- One boundary to set
- How to track progress
- 30-day review date

**Bonus:** Values Decision Framework for making tough choices

**Key Learning:** Values are a compass, not a map—they guide but don't dictate
