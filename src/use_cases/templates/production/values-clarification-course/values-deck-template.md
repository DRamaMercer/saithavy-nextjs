## How to Use Your Values Deck

### Daily Practice

**Morning:**
1. Pull out your values deck
2. Read through all 5 cards
3. Choose ONE value to focus on today
4. Ask: "What's one way I can live this value today?"

**Throughout the Day:**
1. When facing a decision, pull out your deck
2. Ask: "Which values are relevant here?"
3. Choose the option that honors your values

**Evening:**
1. Review your chosen value for the day
2. Reflect: "How did I live this value today?"
3. Acknowledge: "Where did I violate this value?"
4. Plan: "What will I do differently tomorrow?"

### Weekly Practice

**Weekly Review (15 minutes):**
1. Spread out all 5 cards
2. For each value, ask:
   - How did I live this value this week?
   - Where did I violate this value?
   - What's one thing I'll do differently next week?

### Quarterly Practice

**Quarterly Values Review (30-60 minutes):**
1. Review all 5 values
2. Ask: "Are these still my top 5 values?"
3. Update your deck if values have shifted
4. Create new cards for any new values
5. Reflect on how you've grown

## Making Your Deck Portable

### Options:

**1. Credit Card Size**
- Print at 3.375 x 2.125 inches
- Fits in wallet
- Most portable option

**2. Business Card Size**
- Print at 3.5 x 2 inches
- Fits in card holder
- Easy to carry

**3. Index Card Size**
- Standard 3 x 5 or 4 x 6 inches
- More space for writing
- Keep in bag or desk

**4. Digital Version**
- Save as photos on your phone
- Create notes app entry
- Use digital flashcard apps

**5. Creative Options**
- Laminate for durability
- Use different colors for different values
- Add images or symbols that represent each value
- Create a small wooden or metal box for storage

## Values Deck Rituals

### Morning Ritual (5 minutes)

1. Find a quiet space
2. Hold your values deck
3. Slowly read each card
4. Notice how each value feels
5. Choose your focus value for the day
6. Set an intention: "Today I will live [value] by [action]"

### Decision Ritual (2 minutes)

1. When facing a decision, pause
2. Pull out your values deck
3. Ask: "Which values are relevant?"
4. Ask: "Which option honors my values?"
5. Make your choice based on values, not pressure

### Evening Ritual (5 minutes)

1. Pull out your values deck
2. Review your focus value for the day
3. Reflect honestly:
   - How did I live this value?
   - Where did I fall short?
4. Forgive yourself for violations
5. Set tomorrow's intention

### Weekly Ritual (15 minutes)

1. Spread out all 5 cards
2. Light a candle or make tea (create ritual)
3. Review each value one by one
4. Celebrate where you lived your values
5. Acknowledge where you didn't
6. Choose focus areas for next week
7. Write down one action per value

## Troubleshooting

### Problem: "I forget to use my deck"

**Solutions:**
- Set a phone reminder for morning and evening
- Keep your deck visible (desk, nightstand, bag)
- Link it to an existing habit (after coffee, before bed)
- Make it part of your phone check-in routine

### Problem: "My values don't feel relevant to this situation"

**Solutions:**
- Look deeper—there's usually a values component
- Consider long-term impact vs. short-term comfort
- Ask: "Which value is most relevant here?"
- Remember: values are general principles, not specific rules

### Problem: "My values conflict and I can't decide"

**Solutions:**
- Prioritize your values hierarchy
- Ask: "Which value matters most in this season?"
- Consider: "What would future me prefer?"
- Accept: sometimes you'll violate one value to honor another

### Problem: "My values have changed"

**Solutions:**
- This is normal and healthy
- Celebrate your growth
- Update your deck with new values
- Let go of old values that no longer fit
- Remember: values evolve as you evolve

## Remember

**Your values deck is a tool, not a test.**

- You won't always live your values perfectly
- You will violate your values sometimes
- Your values will conflict and compete
- Your values will change over time

**This is all normal.**

**What matters:**
- You know what your values are
- You're trying to live them
- You're honest when you don't
- You keep coming back to them

**Use your deck. Carry it with you. Refer to it often.**

**Let your values be your compass.**

**Lead yourself first.**
