## Format Specifications

### YouTube Shorts

**Technical Specs:**
- **Aspect Ratio**: 9:16 (vertical)
- **Resolution**: 1080x1920 minimum
- **Duration**: Under 60 seconds (optimal: 15-30 seconds)
- **Frame Rate**: 24fps, 30fps, or 60fps
- **File Format**: MP4, MOV, or WEBM
- **File Size**: Under 256GB (practical: under 500MB)

**Shorts Strategy:**
```
Purpose: Discovery and funnel to long-form

Hook (0-3 seconds):
- Visual shock or bold statement
- Pattern interrupt
- Tease full video value

Value (3-60 seconds):
- One clear idea
- Fast-paced delivery
- Text overlays for key points

CTA (end):
- "Watch full video on my channel"
- "Subscribe for Part 2"
- "Link in bio for full guide"
```

**Shorts Hook Templates:**
```
// Teaser Formula
"You won't believe what happened when I {action}..."
"Stop scrolling if you {problem}..."

// Value Tease
"The secret to {outcome} is {unexpected thing}..."
"Here's how I {achievement} in {timeframe}..."

// Controversial Statement
"Everything you know about {topic} is wrong..."
"People think {common belief}, but actually..."
```

### YouTube Long-Form

**Technical Specs:**
- **Aspect Ratio**: 16:9 (horizontal) standard
- **Resolution**: 1920x1080 (1080p) minimum, 4K recommended
- **Duration**: 8-20 minutes (sweet spot), up to 12 hours for live
- **Frame Rate**: 24fps, 30fps, or 60fps
- **File Format**: MP4, MOV, or AVI
- **File Size**: Up to 256GB

**Long-Form Structure:**
```
[0:00-0:30] HOOK
├─ Problem statement or bold claim
├─ Tease what's coming
├─ Establish credibility
└─ Preview value (what they'll learn)

[0:30-1:00] INTRO & CONTEXT
├─ Who you are (brief)
├─ Why this matters
├─ Video roadmap
└─ What to expect

[1:00-8:00] CORE CONTENT (The Meat)
├─ Framework or method
├─ Step-by-step guidance
├─ Examples and proof
├─ Address objections
└─ Real-world application

[8:00-9:30] BONUS VALUE
├─ Pro tips or advanced tactics
├─ Common mistakes to avoid
├─ Resource recommendations
└─ Deeper dives

[9:30-10:00] CTA & OUTRO
├─ Subscribe request
├─ Comment prompt
├─ Related video recommendation
└─ Sign-off
```

## YouTube SEO Strategy

### Title Optimization

**Title Formula:**
```
[Number] [Adjective] [Topic] [Promise/Outcome] - [Timeframe/Condition]

Examples:
- "7 Proven Strategies to Grow Your YouTube Channel in 30 Days"
- "How I Made $10K in One Month (Step-by-Step Tutorial)"
- "The Complete Guide to {Topic} - Beginners Welcome!"
- "Stop Making This {Topic} Mistake (Do This Instead)"
```

**Title Best Practices:**
- 60 characters maximum (before truncation)
- Primary keyword in first 60 characters
- Use numbers when applicable (specificity)
- Power words: "Complete", "Proven", "Ultimate", "Step-by-Step"
- Avoid clickbait (delivers on promise)

### Tag Strategy

**Tag Formula (3-5 tags):**
```
1. Exact match primary keyword
2. Broad category keyword
3. Two related long-tail keywords
4. One branding tag (your channel name)

Example for "how to grow youtube channel":
- "how to grow youtube channel"
- "youtube tips"
- "youtube channel growth strategies"
- "increase youtube subscribers"
- "your channel name"
```

### Chapter/Timestamp Optimization

**Why Chapters Matter:**
- Algorithm understands video structure
- Viewers can jump to relevant sections
- Increases watch time (key ranking signal)
- Shows in search results with timestamps

**Chapter Template:**
```
0:00 - Hook/Title
1:15 - First Major Point
3:45 - Second Major Point
6:20 - Third Major Point
9:00 - Bonus/Advanced Tips
11:30 - Call to Action
12:00 - Outro
```

**Best Practices:**
- Start each chapter at timestamp 0:00, 1:00, 3:00, etc. (algorithm preference)
- Use clear, descriptive chapter titles
- Include primary keywords in chapter titles
- Minimum 3 chapters (preferably 5-8)
- Chapters every 2-4 minutes

## Thumbnail Strategy

### Thumbnail Design Specifications

**Technical Requirements:**
- **Resolution**: 1280x720 (16:9 ratio)
- **File Format**: JPG, PNG, GIF, or BMP
- **File Size**: Under 2MB (under 1MB recommended)
- **Minimum Width**: 640 pixels

### Thumbnail Formulas

```
// Before/After
Split screen: Left = Before, Right = After
Text: "Then vs Now" or "Try This!"

// Face + Text
Your face (expression) + Bold headline text
Text: "This Changed Everything" or "Stop Doing This"

// Pointing/Arrow
Subject pointing to something important
Text: "Look Here" or "This One Thing"

// Comparison
Side-by-side comparison
Text: "Wrong Way" vs "Right Way"

// Number/List
Visual representation of number + text
Text: "7 Tips You Need" or "Step 1 of 3"

// Reaction Shot
Shocked/surprised face + context
Text: "I Can't Believe This Worked"
```

### Thumbnail Best Practices

**Visual Elements:**
- High contrast colors (yellow, red, blue stand out)
- Large, bold text (60pt+ minimum)
- Clear focal point (single subject or object)
- Emotional face (if applicable)
- 3-5 words maximum on thumbnail
- Consistent branding (colors, fonts, style)

**Text Guidelines:**
- Sans-serif fonts (Arial, Impact, Open Sans)
- White text with colored outline (or vice versa)
- 3-5 words maximum
- Complement title, don't repeat it
- Create curiosity gap

**A/B Testing:**
```
Test One Variable at a Time:
1. Headline text
2. Facial expression
3. Color scheme
4. Background image

Run Test:
- Upload 2-3 thumbnail variations
- YouTube will show best performer
- Check analytics after 24-48 hours
- Update to winner after testing
```

## Algorithm Optimization

### Key Ranking Signals (2025)

1. **Click-Through Rate (CTR)** - Most Important
   - Target: 10%+ CTR
   - Driven by: Title + Thumbnail

2. **Average View Duration** - Critical
   - Target: 50%+ of video watched
   - Driven by: Content quality, pacing, editing

3. **Watch Time** - Cumulative
   - Total minutes watched
   - Driven by: Video length + completion rate

4. **Engagement Rate** - Interaction
   - Likes, comments, shares (relative to views)
   - Driven by: Community building, CTAs

5. **Session Time** - Platform Retention
   - How long viewer stays on YouTube
   - Driven by: Playlists, related content

6. **Subscriber Growth** - Long-term Signal
   - Subscribers gained from video
   - Driven by: Value delivery, consistency

### Content Categorization

```
Help YouTube understand your content:

1. Use consistent keywords (titles, descriptions, tags)
2. Create content in series/playlists (builds authority)
3. Stay in your niche (don't pivot randomly)
4. Use cards/end screens (keep viewers on your content)
5. Post consistently (signals reliability)
```

## Common Mistakes to Avoid

1. **Weak thumbnails** - CTR is everything
2. **Misleading titles** - High CTR + low watch time hurts channel
3. **Poor audio** - Viewers will click away immediately
4. **No CTA** - Missing subscriber and engagement opportunities
5. **Inconsistent posting** - Algorithm and audience punishment
6. **Ignoring comments** - Missed community building
7. **No playlists** - Missed watch time opportunity
8. **Jumping between niches** - Confuses algorithm and audience
