## Format Specifications

### Single Tweet

**Character Limits:**
- **Text**: 280 characters maximum
- **URLs**: 23 characters (regardless of actual length)
- **Images**: Up to 4 images (doesn't affect character count)
- **Videos**: Up to 2:20 minutes (doesn't affect character count)
- **Polls**: 25 characters per option, up to 4 options

**Media Specifications:**
- **Images**: JPG, PNG, GIF, WEBP (max 5MB for GIF, 15MB for photos)
- **Videos**: MP4, MOV (up to 512MB for verified users)
- **Resolution**: Recommended 1280x720 minimum (16:9 ratio)

### Twitter Threads

**Structure:**
- **Tweets per thread**: 5-10 optimal (maximum 100+)
- **First tweet**: Hook + thread outline
- **Middle tweets**: Value delivery, one point per tweet
- **Final tweet**: Summary + CTA

**Thread Hook Template:**
```
{Bold statement or hook}

A thread 🧵

1/{total_tweets}
```

## Hashtag Strategy

### Twitter Hashtag Formula (1-2 per tweet)

```markdown
1. **Broad Niche** (1 hashtag)
   - Example: #marketing, #tech, #business

2. **Specific Topic** (1 hashtag, optional)
   - Example: #digitalmarketing, #saas, #productivity

Total: 1-2 hashtags per tweet maximum
```

**Why Fewer Hashtags:**
- Twitter's search is keyword-based, not hashtag-dependent
- Too many hashtags looks spammy
- Focus on keywords in tweet text instead

### Keyword Strategy

```
Include relevant keywords in your tweet text:

"The best productivity apps for entrepreneurs"

Keywords: productivity, apps, entrepreneurs
(These terms will show up in search without hashtags)
```

## Algorithm Optimization

### Key Ranking Signals (2025)

1. **Engagement Rate** (Most Important)
   - Replies > Retweets > Likes > Bookmarks
   - Focus on sparking conversation

2. **Recency**
   - Tweets have short lifespan (minutes to hours)
   - Post when audience is active

3. **Media Type**
   - Video > GIF > Images > Text
   - Native uploads perform better than links

4. **Completion Rate** (For threads)
   - Percentage of thread read
   - Hook critical for first tweet

5. **Author-Audience Relationship**
   - Engage with your audience
   - Reply to comments

### Content Categorization

```
Help Twitter categorize your content:

1. Use consistent keywords (not just hashtags)
2. Stay in your niche (don't pivot randomly)
3. Create content series (builds authority)
4. Engage with similar accounts
5. Post consistently (signals reliability)
```

## Thread Writing Best Practices

### Structure Guidelines

1. **First Tweet is Critical**
   - 50%+ of people decide to read thread based on first tweet
   - Hook must be compelling
   - Include tweet count (e.g., "1/10")

2. **One Idea Per Tweet**
   - Don't cram multiple points
   - Each tweet should stand alone
   - Use line breaks for readability

3. **Internal CTAs**
   - Don't use CTAs in every tweet
   - One CTA at the end is sufficient
   - Use "Continue reading" if thread is very long

4. **Visual Variety**
   - Include images in some tweets
   - Use spacing and formatting
   - Break up text blocks

### Thread Writing Workflow

```
1. Outline thread (all points you want to cover)
2. Write first tweet (hook + tweet count)
3. Write body tweets (one point per tweet)
4. Write final tweet (CTA + summary)
5. Review and edit (remove fluff)
6. Add spacing and formatting
7. Schedule or post immediately
8. Pin first tweet (if performing well)
```

## Common Mistakes to Avoid

1. **Weak hooks** - First 70 characters critical
2. **Too many hashtags** - 1-2 maximum per tweet
3. **Posting links without context** - Add value, don't just drop links
4. **Ignoring replies** - Engagement is everything on Twitter
5. **Inconsistent posting** - Algorithm rewards consistency
6. **Over-promoting** - Provide value first, promote second
7. **Poor thread structure** - Each tweet should stand alone
8. **Not engaging with others** - Twitter is a conversation
