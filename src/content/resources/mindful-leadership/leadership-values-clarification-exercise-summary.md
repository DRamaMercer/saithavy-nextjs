# Leadership Values Clarification Exercise - Quick Reference

## Overview
A comprehensive values clarification system that guides leaders from values discovery through embodied practice in 2-3 hours total.

## What's Included

### 1. Values Discovery (From 100+ Possibilities)
- Complete values inventory (120+ values)
- First narrowing pass (top 15-20)
- Non-negotiable test to identify top 5-7 core values using 5 criteria:
  - **Pain Test:** Would refuse to violate even if it cost you your job
  - **Timeline Test:** Consistent throughout your life/career
  - **Energy Test:** Energizes you when honored
  - **Legacy Test:** Want inscribed on your leadership legacy
  - **Difference Test:** Uniquely distinguishes your leadership

### 2. Values Definition
- Personal definition for each value (not dictionary definitions)
- Emotional response mapping (how it feels when honored/violated)
- Context-specific meaning for YOUR leadership

### 3. Values Integration Assessment
- Current alignment scoring (1-10 scale) for each value
- Specific gap identification with triggers
- Evidence-based rating system

### 4. Decision-Testing Framework
- 5-question test for any significant decision:
  1. Which values are relevant?
  2. If I honored values fully, what would I choose?
  3. If I violated values, what would I choose?
  4. Can I make this decision while staying true to all values?
  5. Five years from now, which choice will make me prouder?
- 5 real-world leadership scenarios to practice:
  - Team member mistakes
  - Difficult news delivery
  - Ethical challenges
  - Work-life tradeoffs
  - Quality vs. deadline pressure

### 5. Values Communication
- Template for articulating values to team
- Conversation guide for one-on-ones
- Team meeting questions for values exploration
- Vulnerability and accountability framing
