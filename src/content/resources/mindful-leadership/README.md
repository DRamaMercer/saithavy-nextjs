# Mindful Leadership Framework - Complete Content Package

## Overview

This is the complete content package for the Mindful Leadership Framework lead magnet. All materials are ready to use as-is—just copy and paste, or customize for your brand voice.

## What's Included

### Core Content Modules

1. **Email Sequence** (5-day nurture sequence)
   - Day 1: Welcome & Self-Awareness Foundation
   - Day 2: Intentional Action (FOCUS method)
   - Day 3: Conscious Communication (CARE listening)
   - Day 4: Team Harmony (Psychological Safety)
   - Day 5: Emotional Mastery & Integration

2. **Self-Awareness Assessment**
   - 50-question comprehensive assessment
   - 5 categories with scoring guides
   - Personal development plan template
   - Reassessment guidance

3. **Intention-Setting Framework (FOCUS Method)**
   - Complete FOCUS framework explanation
   - Real-world examples
   - Decision-making worksheet template
   - Common decision traps and how to avoid them
   - Quick version for faster decisions

4. **Communication Templates** (12 templates)
   - Giving difficult feedback (SBI model)
   - Requesting behavior change
   - Addressing missed deadlines
   - Resolving team conflict (CARE method)
   - Delivering bad news
   - Asking for resources
   - Setting boundaries
   - Giving recognition
   - Conducting 1:1 meetings
   - Apologizing when wrong
   - Delegating effectively
   - Asking for feedback

5. **Team Culture Building Playbook**
   - Culture assessment and diagnostics
   - Step-by-step building process
   - 6 team activities with full instructions
   - Communication rhythm templates
   - Troubleshooting guide for common challenges
   - Quick-start implementation plan

6. **Conflict Resolution Guide (CARE Method)**
   - Understanding conflict types
   - Complete CARE framework
   - 5 scripts for common conflict situations
   - Prevention strategies
   - Advanced conflict skills
   - Practice exercises

### Bonus Materials

7. **Bonus Materials Collection**
   - 30-Day Mindful Leadership Challenge
   - Quarterly Leadership Review template
   - Leadership Journal Prompts (daily & weekly)
   - Meeting Facilitation Toolkit
   - Decision-Making Log
   - Values Clarification Exercise
   - Quick-Start Implementation Checklist
   - Crisis Leadership Checklist
   - Book Recommendations
   - Resource Library
