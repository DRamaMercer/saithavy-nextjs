## Quick Reference Guides

### AI Tool Selection Guide

| Tool | Best For | Pricing | Key Features | Limitations |
|------|----------|---------|--------------|-------------|
| **ChatGPT** | General text, brainstorming | Free/$20/mo | Conversational, versatile | Knowledge cutoff, hallucinations |
| **Claude** | Long-form content, analysis | Free/$20/mo | Large context, nuanced | Less common integrations |
| **Jasper** | Marketing copy | $49+/mo | Templates, brand voice | Expensive for basic use |
| **Copy.ai** | Quick copy variations | $36+/mo | Fast iterations | Less control |
| **Midjourney** | Images | $10-$60/mo | High quality | Text-to-image only |
| **Zapier** | Workflow automation | $20+/mo | Connects apps | Learning curve |

**Selection Framework**:
1. What's your primary use case?
2. What's your budget?
3. What integrations do you need?
4. What's your team's technical comfort?

**Role** ✅
- [ ] Defined who AI should be
- [ ] Relevant expertise specified

## Troubleshooting Guides

### Common Issues & Solutions

**Problem**: Generic, unhelpful AI responses

**Solutions**:
1. Add more specific context
2. Define the role more clearly
3. Provide examples of desired output
4. Break into smaller, specific tasks
5. Use follow-up prompts to refine

**Problem**: AI hallucinates (makes things up)

**Solutions**:
1. Ask AI to cite sources
2. Provide source material to work from
3. Verify factual claims before using
4. Use AI for synthesis, not facts
5. Add "only use information provided" constraint

**Problem**: Output doesn't match brand voice

**Solutions**:
1. Provide brand voice examples
2. Describe tone explicitly (professional, friendly, etc.)
3. Share previous successful content
4. Iterate and refine based on outputs
5. Create brand voice prompt template

**Problem**: AI won't do what you want

**Solutions**:
1. Check if tool has the capability
2. Break complex tasks into steps
3. Try a different AI tool
4. Rephrase your request
5. Ask AI "Can you help me with...?"

**Problem**: Security/compliance concerns

**Solutions**:
1. Remove sensitive data before inputting
2. Use enterprise versions with data protection
3. Create policy on what can/can't be shared
4. Sanitize outputs before using
5. Get legal/compliance review
