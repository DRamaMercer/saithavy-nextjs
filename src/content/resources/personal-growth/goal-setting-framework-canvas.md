# Goal Setting Framework Canvas
## A Visual Framework for Meaningful Personal Goals

---

*A Strategic, Values-Aligned Approach to Setting Goals That Actually Matter—and Actually Happen*

---

## How to Use This Canvas

This canvas is designed to help you set goals that are:
- **Aligned** with your core values
- **Meaningful** to you (not others)
- **Achievable** through consistent action
- **Sustainable** over the long term

**Recommended Process:**
1. Complete the canvas in order (Section 1 → 7)
2. Be honest with yourself
3. Review and revise quarterly
4. Keep it visible as a daily reminder

### The 3-Part Goal Statement

Use this formula for each goal:

"I will [MEASURABLE OUTCOME] by [DATE] by [ACTIONS I'LL TAKE].

This matters to me because [VALUES/MEANING].

When I achieve this, I will feel [EMOTION]."

## Quick Reference: Goal Setting Checklist

Use this checklist before finalizing any goal:

- [ ] Aligned with my core values
- [ ] Specific and measurable
- [ ] Has a clear deadline
- [ ] Realistic but stretching
- [ ] Excites and motivates me
- [ ] Broken into actionable steps
- [ ] Obstacles identified and planned for
- [ ] Tracking method in place
- [ ] Review schedule established
- [ ] Celebration milestones defined

## Final Thoughts

> "A goal without a plan is just a wish." — Antoine de Saint-Exupéry

But a goal without meaning is just busywork.

The most powerful goals come from the intersection of:
- **Who you are** (your values)
- **Who you want to become** (your identity)
- **What you want to create** (your outcomes)

Use this canvas to set goals that matter. Then pursue them with consistency, flexibility, and self-compassion.

You've got this.
