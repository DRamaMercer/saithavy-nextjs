## Content Optimization Checklist

### On-Page SEO (Per Resource)

#### Title Tags (60 characters max)
- [ ] Include primary keyword
- [ ] Include "Free" or "Download" (value proposition)
- [ ] Brand name (if space permits)
- [ ] Action-oriented language

**Example Template**: `[Primary Keyword] | Free [Type] Download`

#### Meta Descriptions (155 characters max)
- [ ] Include primary keyword
- [ ] Clear value proposition
- [ ] Call-to-action (Download, Get, Access)
- [ ] Emotional hook or benefit

**Example Template**: `[Benefit]. Free [Resource Name] for [Target Audience].`

#### H1 Tags
- [ ] One H1 per page (resource title)
- [ ] Include primary keyword
- [ ] Descriptive and compelling
- [ ] Under 70 characters

#### H2/H3 Tags
- [ ] Include secondary keywords
- [ ] Natural language flow
- [ ] Descriptive of content sections
- [ ] Hierarchical structure

#### URL Structure
- [ ] Hyphenated words
- [ ] Include primary keyword
- [ ] Short and descriptive
- [ ] Lowercase only

**Example**: `/resources/mindful-leadership-reflection-journal`

#### Image Optimization
- [ ] Descriptive file names (keyword-rich)
- [ ] Alt text with keywords
- [ ] Compressed for speed (under 100KB)
- [ ] Responsive images

#### Content Optimization
- [ ] Primary keyword in first 100 words
- [ ] Secondary keywords naturally throughout
- [ ] Keyword density: 1-2% (natural)
- [ ] Readability: 8th grade level (Flesch-Kincaid)
- [ ] Scannable formatting (bullet points, headers)
- [ ] Internal links (2-3 minimum)
- [ ] External links to authority sources (1-2)

#### Technical SEO
- [ ] Mobile-responsive design
- [ ] Page load speed under 3 seconds
- [ ] SSL certificate (HTTPS)
- [ ] Schema markup (Product, Article)
- [ ] Canonical tags
- [ ] XML sitemap inclusion
- [ ] Robots.txt optimization
