### Week 1: Foundation
- [ ] Choose ESP (see Implementation Guide)
- [ ] Authenticate domain (critical for deliverability)
- [ ] Set up signup forms
- [ ] Create lead magnet
- [ ] Build welcome sequence (5-7 emails)

### Week 2: First Automation
- [ ] Launch welcome automation
- [ ] Set up basic segments (customers, leads, inactive)
- [ ] Create abandoned cart sequence (if e-commerce)
- [ ] Test with small segment

### Week 3: Revenue Automations
- [ ] Build post-purchase sequence
- [ ] Set up re-engagement sequence
- [ ] Launch to full list
- [ ] Start A/B testing (subject lines first)

### Week 4: Optimization
- [ ] Clean list (remove bounces, segment inactives)
- [ ] Set up metrics dashboard
- [ ] Review first month metrics
- [ ] Plan next automations
