# Saithavy Resources - Brand Style Guide

## Brand Identity

- **Name**: Saithavy Sengehanh
- **Tagline**: Transform with Intention. Empower Your Journey.
- **Website**: Resources library for leaders, entrepreneurs, remote workers, growth-minded individuals

## Color Palette

| Name | Hex | Usage |
|------|-----|-------|
| Navy | #1B263B | Primary backgrounds, headings, text, dark sections |
| Terracotta | #E07A5F | Primary CTAs, accents, featured elements, buttons |
| Mustard | #F4A261 | Secondary accents, highlights, ratings stars |
| Sage | #A8DADC | Borders, backgrounds, subtle highlights, cards |
| Cool Gray | #5E6472 | Body text, secondary elements, icons |

### Color Usage by Element

**Primary CTA Buttons**
- Default: `bg-[#E07A5F] hover:bg-[#c4684f]`
- Text: white
- Border radius: 8px (rounded-lg)

**Secondary CTA Buttons**
- Default: `bg-[#1B263B] hover:bg-[#2D3A50]`
- Text: white
- Border radius: 8px

**Outline Buttons**
- Border: `border-2 border-[#E07A5F]`
- Text: `text-[#E07A5F]`
- Hover: `hover:bg-[#E07A5F] hover:text-white`

**Card Backgrounds**
- Default: white
- Hover: subtle shadow increase, slight translate-y

**Section Backgrounds**
- Primary: Navy gradient `bg-gradient-to-br from-[#1B263B] to-[#2D3A50]`
- Secondary: Sage tint `bg-[#A8DADC]/20`
- Tertiary: Terracotta tint `bg-[#E07A5F]/10`

## Typography

### Font Families

- **Headings**: Poppins (Google Fonts)
  - Weights: 400 (regular), 600 (semibold), 700 (bold)
  
- **Body**: Roboto (Google Fonts)
  - Weights: 300 (light), 400 (regular), 500 (medium)

### Font Sizes

| Element | Size | Weight | Line Height |
|---------|------|--------|-------------|
| H1 | 48-60px (text-5xl, text-6xl) | 700 | 1.2 |
| H2 | 36px (text-4xl) | 700 | 1.2 |
| H3 | 24px (text-xl) | 600 | 1.3 |
| Body Large | 20px (text-xl) | 400 | 1.6 |
| Body | 16px (text-base) | 400 | 1.6 |
| Small | 14px (text-sm) | 400 | 1.5 |
| Caption | 12px (text-xs) | 500 | 1.4 |

## Voice & Tone

### Key Messages

- "Transform with Intention. Empower Your Journey."
- "Lead with Clarity and Intention"
- "Work Smarter with AI"
- "Embrace Your Evolution"
- "Thrive Anywhere"
- "Rise Through Challenges"

### Tone Attributes

- **Professional** yet approachable
- **Expert** guidance with empathy
- **Action-oriented** prompts and exercises
- **Transformational** language
- **Clear** value propositions

### Writing Style

- Use active voice
- Keep sentences under 25 words
- Start with benefit-driven headlines
- Use bullet points for lists
- Include clear CTAs

| Breakpoint | Width |
|-------------|-------|
| Mobile | < 640px |
| Tablet | 640px - 1024px |
| Desktop | > 1024px |

## Accessibility

- Color contrast: Minimum 4.5:1 for text
- Focus states: Visible outline on interactive elements
- Alt text: Descriptive for all images
- Semantic HTML: Proper heading hierarchy

## Animation Guidelines

- Subtle, purposeful animations only
- Hover states: translate-y, shadow increases
- Page transitions: fade-in, slide-up
- Duration: 200-300ms for micro-interactions
- Avoid: Flashy, distracting animations
