### Color Usage Guidelines

**DO:**
- Use category accent colors consistently
- Ensure minimum 4.5:1 contrast ratio for normal text

**DON'T:**
- Use accent colors for large areas of background
- Use colors that don't meet WCAG AA standards
- Overuse bright colors (strategic highlighting only)
- Use color as the only indicator of meaning
- Use similar colors for different categories

## Typography System

### Font Families

**Primary Font Family**
- **Name:** Inter
- **Usage:** Headings, body text, UI elements
- **Weights:** 400 (Regular), 500 (Medium), 600 (Semibold), 700 (Bold)
- **Fallback:** system-ui, -apple-system, BlinkMacSystemFont, sans-serif

**Secondary Font Family**
- **Name:** Open Sans
- **Usage:** Alternative to Inter, printed materials
- **Weights:** 400 (Regular), 600 (Semibold), 700 (Bold)
- **Fallback:** sans-serif

**Monospace Font Family**
- **Name:** JetBrains Mono
- **Usage:** Code, technical content, data
- **Weights:** 400 (Regular), 500 (Medium)
- **Fallback:** 'Courier New', monospace

### Type Scale

**Digital (Web/UI):**
- **H1 (Page Title):** 40px / 48px (Bold, 700)
- **H2 (Section Title):** 32px / 40px (Semibold, 600)
- **H3 (Subsection):** 24px / 32px (Semibold, 600)
- **H4 (Card Title):** 20px / 24px (Semibold, 600)
- **Body Large:** 18px (Regular, 400)
- **Body Base:** 16px (Regular, 400)
- **Body Small:** 14px (Regular, 400)
- **Caption:** 12px (Regular, 400)

**Print (PDF/Documents):**
- **Document Title:** 36pt (Bold)
- **Section Title:** 28pt (Semibold)
- **Subsection:** 20pt (Semibold)
- **Body Text:** 11-12pt (Regular)
- **Captions:** 9pt (Regular)

### Typography Guidelines

**DO:**
- Use consistent font weights for hierarchy
- Maintain line height 1.4-1.6 for body text
- Use letter-spacing -0.02em for headings
- Limit line length to 60-80 characters
- Use proper heading hierarchy (H1→H2→H3→H4)

**DON'T:**
- Use more than 3 font sizes in one section
- Use font sizes smaller than 12px for body text
- Use all caps for body text (reduces readability)
- Use underlines for non-links
- Use italic for large blocks of text

## Logo Usage

### Logo Specifications

**Primary Logo**
- **Type:** Lockup (icon + text)
- **Icon:** Category icon or brand mark
- **Text:** "Sai Resources"
- **Font:** Inter, Semibold (600)
- **Size:** Flexible, maintain proportions
- **Clear Space:** Logo height × 0.5 on all sides

**Logo Icon Only**
- **Usage:** Favicon, app icons, avatars
- **Min Size:** 32x32px
- **Preferred Size:** 256x256px

**Logo Color Variations**
- **Full Color:** On light backgrounds
- **White:** On dark backgrounds (when color doesn't work)
- **Monochrome:** When color reproduction not available

### Logo Placement

**Preferred Locations:**
- Top left (header navigation)
- Center (home page)
- Footer (branding)

**Minimum Size:**
- Width: 120px
- Height: Auto (maintain aspect ratio)

**Clear Space:**
- Maintain clear space equal to logo height on all sides
- No text, graphics, or other elements in clear space

### Logo Don'ts

**DON'T:**
- Stretch or distort the logo
- Change logo colors (use official variations)
- Rotate the logo
- Add effects (drop shadows, gradients, etc.)
- Place logo on busy backgrounds
- Use logo smaller than minimum size
- Crowd logo with other elements

## Visual Style

### Design Principles

**1. Clarity Over Complexity**
- Clean, uncluttered layouts
- Plenty of white space
- Clear visual hierarchy
- Intuitive navigation

**2. Accessibility First**
- WCAG 2.1 AA compliant
- High contrast ratios
- Readable font sizes
- Screen reader friendly

**3. Category Consistency**
- Category-specific accent colors
- Consistent iconography
- Themed imagery
- Unified typography

**4. Mobile-First**
- Responsive design
- Touch-friendly targets (44x44px minimum)
- Readable on small screens
- Optimized images

### Layout Guidelines

**Spacing System**
- **Unit:** 4px base unit
- **XS:** 4px (0.25rem)
- **SM:** 8px (0.5rem)
- **MD:** 16px (1rem)
- **LG:** 24px (1.5rem)
- **XL:** 32px (2rem)
- **2XL:** 48px (3rem)

**Grid System**
- **Columns:** 12-column grid
- **Gutter:** 16px (1rem)
- **Max Width:** 1200px (75rem)
- **Margins:** 16px (1rem) on mobile, 24px (1.5rem) on desktop

**Card Design**
- **Border Radius:** 8px
- **Shadow:** 0 1px 3px rgba(0,0,0,0.12), 0 1px 2px rgba(0,0,0,0.24)
- **Padding:** 16px (1rem)
- **Background:** #FFFFFF
- **Border:** 1px solid #E5E7EB

**Buttons**
- **Primary:** Category accent color, white text
- **Secondary:** #1B263B background, white text
- **Border Radius:** 6px
- **Padding:** 12px 24px (0.75rem 1.5rem)
- **Font:** Semibold (600)
- **Transition:** 0.2s ease-in-out

## Voice and Tone

### Brand Voice Characteristics

**Empowering:**
- "You can do this"
- "Here's how to succeed"
- "You have what it takes"

**Clear:**
- Simple, direct language
- Avoid jargon or explain it
- One idea per sentence

**Actionable:**
- Specific steps
- Concrete examples
- Real-world application

**Supportive:**
- Understanding tone
- Acknowledges challenges
- Celebrates progress

### Writing Style

**DO:**
- Use second person ("you")
- Active voice ("Complete this step")
- Present tense
- Short paragraphs (2-3 sentences)
- Conversational but professional

**DON'T:**
- Use third person ("one should")
- Passive voice ("it is recommended")
- Future tense unnecessarily
- Long paragraphs
- Academic language

### Content Voice Examples

**Mindful Leadership:**
> "Take a moment to reflect on your leadership journey. This journal will help you identify your strengths and growth areas."

**AI + Automation:**
> "Save time by automating repetitive tasks. This template shows you exactly which workflows to automate first."

**Personal Growth:**
> "Growth isn't always easy, but it's always worth it. This framework helps you navigate challenges with resilience."

**Remote Work:**
> "Create a workspace that works for you. This checklist ensures you have everything you need for productive remote work."

**Overcoming Adversity:**
> "You're stronger than you know. This resilience map has helped thousands navigate difficult seasons."
