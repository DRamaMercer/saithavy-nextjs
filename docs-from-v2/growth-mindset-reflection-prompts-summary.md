## Key Features

### Progressive Difficulty Structure
**Week 1: Self-Awareness** (Foundation)
- Notice your mindset
- Identify fixed mindset triggers
- Learn the Rebuild Formula
- Start simple, build awareness

**Week 2: Challenge** (Pushing Comfort Zones)
- Embrace difficulty
- View discomfort as growth data
- Take small risks
- Build challenge muscles

**Week 3: Learning** (Process Focus)
- Focus on process, not outcome
- Celebrate mistakes as learning
- Receive feedback constructively
- Identify learning styles

**Week 4: Resilience** (Integration)
- Bounce back from setbacks
- Build support networks
- Reframe past experiences
- Create sustainable practices

### The Rebuild Formula
5-step process for reframing fixed mindset thoughts:
1. **NOTICE**: "I'm having a fixed mindset thought"
2. **NAME**: "The thought is: [specific thought]"
3. **NORMALIZE**: "This is my brain trying to protect me from discomfort"
4. **REFRAME**: "A growth mindset would say: [new perspective]"
5. **ACT**: "What's one small action I can take from this new perspective?"

### Real Examples Included
- Perfectionist → Progress-Maker
- Comparison → Collaboration
- Victim → Agent
- Identity-Based Avoidance → Skill-Building
- All-or-Nothing → Experimentation
