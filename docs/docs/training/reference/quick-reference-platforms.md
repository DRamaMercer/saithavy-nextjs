# Platform Quick Reference Guide

## TikTok

### Key Specs
- **Video Length:** 15-60 seconds (optimal)
- **Dimensions:** 1080x1920 (9:16)
- **Format:** MP4, MOV
- **Max File Size:** 287.6 MB

### Algorithm Focus
- Watch time and completion rate
- Engagement velocity (speed of engagement)
- Shares and saves
- Video completion

### Posting Strategy
- **Frequency:** 2-4x daily
- **Peak Times:** 12-3 PM, 7-10 PM
- **Hashtags:** 3-5 total (mix trending + niche)
- **Audio:** Use trending sounds

### Content Types
- Dance challenges
- Educational tutorials
- Behind-the-scenes
- Trend participation
- Storytelling

### Performance Benchmarks
- **Views:** 10K+ for good content
- **Engagement:** 5-10%
- **Shares:** Key metric for virality
- **Completion:** >70% at 3 seconds

## Instagram

### Key Specs
- **Feed Posts:** 1080x1080 (1:1) or 1080x1350 (4:5)
- **Reels:** 1080x1920 (9:16), 15-30 seconds
- **Stories:** 1080x1920 (9:16), 15 seconds max
- **Carousels:** Up to 10 slides

### Algorithm Focus
- Engagement (likes, comments, saves, shares)
- Relationship with followers
- Timeliness
- Profile visits

### Posting Strategy
- **Feed:** 1x daily, 11 AM-1 PM or 7-9 PM
- **Reels:** 1-2x daily
- **Stories:** 3-5x daily
- **Hashtags:** 10-15 (mix broad, niche, branded)

### Content Types
- Aesthetic photos
- Reels (remix TikToks)
- Behind-the-scenes
- User-generated content
- Educational carousels

### Performance Benchmarks
- **Engagement:** 3-5% good, 5%+ excellent
- **Reach:** 10-20% of followers
- **Saves:** Key for algorithm
- **Profile Visits:** Growth indicator

---

## YouTube

### Key Specs
- **Long-form:** 10-20 minutes optimal
- **Shorts:** 15-60 seconds
- **Dimensions:** 1920x1080 (16:9)
- **Thumbnail:** 1280x720 (16:9)

### Algorithm Focus
- Watch time (primary)
- Click-through rate (CTR)
- Engagement velocity
- Session time

### Posting Strategy
- **Long-form:** 3-5x weekly, 2-4 PM or 6-9 PM
- **Shorts:** 1-2x daily
- **Premiere:** Build anticipation
- **Consistency:** Same days/times

### Content Types
- Tutorials
- Vlogs
- Educational
- Entertainment
- Reviews

### Performance Benchmarks
- **CTR:** 5-10% good, 10%+ excellent
- **AVD (Average View Duration):** 50%+ good
- **Watch Time Hours:** Key for monetization
- **Subscribers:** 1% conversion per 1000 views

---

## Twitter/X

### Key Specs
- **Text:** 280 characters
- **Video:** 2:20 max (140 seconds)
- **Images:** Up to 4
- **Polls:** 4 options, 7 days max

### Algorithm Focus
- Engagement speed
- Recency
- Media richness
- Author authority

### Posting Strategy
- **Frequency:** 5-10x daily
- **Timing:** Continuous (real-time)
- **Threads:** Tell longer stories
- **Hashtags:** 1-3 relevant only

### Content Types
- Threads (storytelling)
- Hot takes/opinions
- Breaking news
- behind-the-scenes
- Polls and questions

### Performance Benchmarks
- **Engagement:** 1-2% good
- **Impressions:** 1K+ per tweet (baseline)
- **Retweets:** Virality metric
- **Replies:** Community building

## LinkedIn

### Key Specs
- **Posts:** 3,000 characters
- **Video:** 10 minutes max
- **Documents:** PDF, PPT
- **Images:** 1200x627 (recommended)

### Algorithm Focus
- Engagement dwell time
- Comments (especially long ones)
- Shares
- Profile clicks

### Posting Strategy
- **Frequency:** 1x daily
- **Peak Times:** 8-10 AM, 5-6 PM
- **Hashtags:** 3-5 professional
- **Consistency:** Build authority

### Content Types
- Thought leadership
- Industry insights
- Case studies
- Professional achievements
- Educational content

### Performance Benchmarks
- **Engagement:** 2-3% good, 5%+ excellent
- **Views:** 500+ for good content
- **Comments:** Quality over quantity
- **Followers:** 10% monthly growth

## Pinterest

### Key Specs
- **Pins:** 1000x1500 (2:3) optimal
- **Video:** 15-30 seconds
- **Carousels:** Up to 5 slides
- **Idea Pins:** Multi-page

### Algorithm Focus
- Pin quality
- Engagement (saves especially)
- Freshness
- Keyword relevance

### Posting Strategy
- **Frequency:** 3-5x daily
- **Peak Times:** 8-11 PM
- **Hashtags:** Not used (keywords instead)
- **Consistency:** Pin regularly

### Content Types
- Infographics
- Product pins
- How-to guides
- Inspirational quotes
- Collection boards

### Performance Benchmarks
- **Impressions:** 10K+ monthly good
- **Saves:** Primary metric
- **Clicks:** Traffic driver
- **Outbound Clicks:** Conversion metric

---

## Facebook

### Key Specs
- **Posts:** 63,206 characters
- **Video:** 240 minutes max
- **Images:** Recommended 1200x630
- **Live:** Up to 8 hours

### Algorithm Focus
- Engagement (comments especially)
- Shares
- Relationships
- Content type preference

### Posting Strategy
- **Frequency:** 1x daily
- **Peak Times:** 1-4 PM
- **Hashtags:** 1-2 (minimal impact)
- **Mix:** Content types

### Content Types
- Community questions
- Educational posts
- Entertainment
- User-generated content
- Live video

### Performance Benchmarks
- **Reach:** 10-20% of page followers
- **Engagement:** 3-5% good
- **Shares:** Viral metric
- **Comments:** Community building

## Hashtag Strategy by Platform

### TikTok
- **Count:** 3-5 total
- **Mix:** 1-2 trending + 2-3 niche
- **Placement:** In caption
- **Research:** Trending page

### Instagram
- **Count:** 10-15 (or 30 if in stories)
- **Mix:** 3 broad + 5 niche + 2 branded + 3 trending
- **Placement:** Caption or first comment
- **Research:** Explore page, competitors

### YouTube
- **Count:** 3-5 tags in metadata
- **Focus:** Search volume, relevance
- **Placement:** Video settings
- **Research:** YouTube search suggestions

### Twitter
- **Count:** 1-3 maximum
- **Focus:** Highly relevant, searchable
- **Placement:** In tweet
- **Research:** Trending topics

### LinkedIn
- **Count:** 3-5
- **Focus:** Industry-specific, professional
- **Placement:** End of post
- **Research:** LinkedIn search, industry leaders

### Pinterest
- **Count:** Not used (keywords instead)
- **Focus:** SEO keywords in description
- **Placement:** Pin description
- **Research:** Pinterest search, guided search

### Facebook
- **Count:** 1-2 (minimal impact)
- **Focus:** Highly relevant only
- **Placement:** In post
- **Research:** Facebook search
