## Learning Path

### Recommended Order

#### Phase 1: Foundation (Week 1-4)
1. **Start with Twitter/X** (fastest learning curve)
2. **Add Pinterest** (visual fundamentals)
3. **Begin Instagram** (platform-optimized content)

#### Phase 2: Intermediate (Week 5-12)
4. **Master TikTok** (short-form video)
5. **Deepen Instagram** (all formats)
6. **Add Facebook** (community building)

#### Phase 3: Advanced (Week 13+)
7. **Conquer YouTube** (long-form strategy)
8. **Dominate LinkedIn** (B2B expertise)

### Platform Pairing Strategy

**Compatible Pairs (can learn simultaneously):**
- TikTok + Instagram Reels (similar formats)
- Twitter + LinkedIn (text-based)
- Pinterest + Instagram (visual focus)
- YouTube + Facebook (long-form video)

**Sequential Learning (learn one after another):**
- Twitter → TikTok → YouTube
- Pinterest → Instagram → Facebook
- LinkedIn → Twitter → Facebook

## Common Platform Mistakes

### Format Mistakes
- Wrong dimensions
- Poor quality video/audio
- Incorrect aspect ratios
- File size issues
- Encoding problems

### Timing Mistakes
- Posting during low-traffic times
- Ignoring timezone differences
- Inconsistent posting schedules
- Missing peak windows
- Over-posting or under-posting

### Content Mistakes
- Not platform-native
- Ignoring algorithm preferences
- Poor hook structure
- Weak CTAs
- Inconsistent branding

### Engagement Mistakes
- Buying followers
- Using engagement pods
- Ignoring comments
- Generic responses
- Over-automation
